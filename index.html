Que es java script?

JavaScript es un lenguaje de programación de alto nivel, interpretado y de propósito general que se utiliza principalmente para agregar interactividad y funcionalidad a las páginas web. Es uno de los tres pilares de las tecnologías web junto con HTML (Hypertext Markup Language) y CSS (Cascading Style Sheets). JavaScript fue creado originalmente por Brendan Eich en 1995 mientras trabajaba en Netscape Communications Corporation. Desde entonces, ha experimentado un tremendo crecimiento y se ha convertido en uno de los lenguajes de programación más populares del mundo.

Ejemplos de JavaScript en HTML

Example.

// Declaración de variables con let

let mensaje = "Hola, mundo!";
console.log(mensaje); // Salida: Hola, mundo!

// Las variables let pueden ser reasignadas

mensaje = "¡Hola, amigos!";
console.log(mensaje); // Salida: ¡Hola, amigos!

// Declaración de variables con const

const PI = 3.14159;
console.log(PI); // Salida: 3.14159

// Las variables const no pueden ser reasignadas
// Esto lanzará un error
// PI = 3;
// Sin embargo, las variables const deben ser inicializadas al momento de la declaración
// Esto lanzará un error
// const nombre;
// Las variables const también pueden ser objetos, y aunque sus propiedades pueden cambiar, la asignación a la variable en sí no puede cambiar.

const persona = {
nombre: "Juan",

};
